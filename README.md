## Project navigation

| Shortcut | Description                         |
| -------- | ----------------------------------- |
| ⌘⇧O      | Open quickly                        |
| ⌘⇧J      | Show current file project navigator |
| ⌃6       | Show document items                 |
| ⌥⌘W      | Close other tabs                    |
| ⌘W       | Close current tab                   |

## File manipulation

| Shortcut | Description            |
| -------- | ---------------------- |
| ⌘⌥[      | Move current line up   |
| ⌘⌥]      | Move current line down |
| ⌃I       | Reindent               |
| ⌘A       | Select all             |
| ⌘S       | Save current file      |

## Testing

| Shortcut | Description              |
| -------- | ------------------------ |
| ⌘U       | Run all tests            |
| ⌃⌥⌘G     | Run previous tests again |

## Refactoring

| Shortcut | Description       |
| -------- | ----------------- |
| ⇧⌘A      | Open code actions |
