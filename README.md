## Project navigation

| Shortcut | Description         |
| -------- | ------------------- |
| ⌘O       | Open quickly        |
| ⌃6       | Show document items |

## File manipulation

| Shortcut | Description            |
| -------- | ---------------------- |
| ⌘⌥[      | Move current line up   |
| ⌘⌥]      | Move current line down |
| ⌃I       | Reindent               |
| ⌘A       | Select all             |
| ⌘S       | Save current file      |
